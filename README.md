# The Pseudoartist Profile Card
A simple HTML page script for a profile card. Feel free to use

## Projects Used: 
 * Bootstrap Icons

## How to use: 
1. Change the HTML title to index.html
2. Change the profile name to your preferred name
3. Change the profile image (if needed)
4. Edit the list items (name, link, etc)
5. Publish with your preferred HTML host (GitHub Actions, AWS S3 Bucket)
6. Copy the link and paste it into the appropriate applications
